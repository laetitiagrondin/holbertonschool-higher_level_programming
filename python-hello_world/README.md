# Python Scripts - Project Folder

## Description
This folder contains Python scripts that demonstrate basic Python programming concepts such as:
- Printing text
- Using variables and f-strings
- String manipulation and slicing
- Concatenation of strings
- Importing modules

*All scripts are executable, start with the shebbang '#!/usr/bin/python3',
and are intended to be run on Ubuntu 20.04 LTS using Python 3.8.\*.
All code follows PEP8 style guidelines checked with 'pycodestyle' 2.7.\*.*

---

## Files and Usage

| File | Description | Usage |
|------|-------------|-------|
| '2-print.py' | Prints a specific sentence with a leading quote | ./2-print.py |
| '3-print_number.py' | Prints the value of a number using a variable and f-string | ./3-print_number.py |
| '4-print_float.py' | Prints a float number with 2 decimal places | ./4-print_float.py |
| '5-print_string.py' | Prints a string 3 times and the first 9 characters | ./5-print_string.py |
| '6-concat.py' | Concatenates two strings and prints a welcome message | ./6-concat.py |
| '7-edges.py' | Prints the first 3, last 2 and middle of a string | ./7-edges.py |
| '8-concat_edges.py' | Concatenates specific parts of a long string to form a new sentence | ./8-concat_edges.py |
| '9-easter_egg.py' | Prints "The Zen of Python" by importing the 'this' module | ./9-easter_egg.py |

---

## Contraints
- All files **must be executable** ('chmod +x filename.py')
- All files **end with a new line**
- The first line of every file must be exactly: **#!/usr/bin/python3**
